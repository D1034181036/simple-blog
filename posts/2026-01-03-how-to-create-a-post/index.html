如何新增一篇文章

2026-01-03

前言

本篇文章將教你如何在這個部落格中新增一篇文章。只需要簡單的幾個步驟，就能輕鬆發布新內容！

步驟概覽

  1. 複製一份資料夾：選擇 posts/ 中的任一文章資料夾並複製
  2. 修改資料夾名稱：重新命名為 YYYY-MM-DD-文章標題 格式
  3. 修改文章內容：編輯 index.html 中的標題、日期和內容
  4. 更新文章列表：在 posts/index.html 中加入新文章的連結

步驟一：複製一份資料夾

首先，找到 posts/ 資料夾中的任一篇現有文章資料夾，例如：

  • 2026-01-01-first-post/
  • 2026-01-02-html-cheat-sheet/

選擇其中一個資料夾，將整個資料夾複製一份。

步驟二：修改資料夾名稱

將複製的資料夾重新命名，格式為：YYYY-MM-DD-文章標題

例如：

  • 2026-01-03-how-to-create-a-post

注意：資料夾名稱建議使用英文和連字號，避免使用空格和特殊符號。

步驟三：修改文章內容

進入新建立的資料夾，打開 index.html 檔案，修改以下內容：

1. 修改標題（title）

在 <head> 區塊中，找到 <title> 標籤並修改：

<title>你的文章標題</title>

2. 修改文章標題（h2）

在 <article> 區塊中，找到 <h2> 標籤並修改：

<h2>你的文章標題</h2>

3. 修改日期

找到日期標籤並修改為你的文章日期：

<p><small>2026-01-03</small></p>

4. 修改文章內容

編輯 <article> 標籤內的所有內容，加入你想要的文字、段落、列表等。

你可以參考 HTML 常用標籤 來使用各種 HTML 元素。

選用：如何放入圖片

在文章中加入圖片非常簡單，請按照以下步驟操作：

1. 準備圖片檔案

將你的圖片檔案（如 image.jpg、photo.png、img.webp 等）放入文章資料夾中，與 index.html 放在同一層。

2. 使用 img 標籤

在 HTML 檔案中，使用 <img> 標籤來顯示圖片：

<img src="你的圖片檔名.jpg" alt="圖片說明">

3. 圖片屬性說明

  • src：圖片檔案的路徑，如果圖片在同一資料夾中，直接寫檔名即可
  • alt：圖片的替代文字，當圖片無法顯示時會顯示此文字，也有助於無障礙閱讀
  • width/height：可選的尺寸屬性，例如 width="500" 可以設定圖片寬度

步驟五：加入文章連結到列表

完成文章撰寫後，需要將新文章加入到文章列表中，讓訪客能夠找到並閱讀你的文章。

1. 開啟文章列表檔案

找到並開啟 posts/index.html 檔案。

2. 加入新文章連結

在 <ul> 列表中，加入你的新文章。格式如下：

<li>
    <time>2026-01-03</time>
    <a href="2026-01-03-how-to-create-a-post/index.html">如何新增一篇文章</a>
</li>

3. 注意事項

  • 新文章應該放在列表的最上方（最新的文章在最前面）
  • <time> 標籤中填入文章日期
  • href 屬性要填入正確的資料夾路徑和檔名
  • <a> 標籤中的文字就是顯示在列表中的文章標題

小提示

  • 記得儲存所有修改過的檔案
  • 可以在瀏覽器中開啟 index.html 預覽效果
  • 養成良好的命名習慣，使用有意義的資料夾名稱和圖片檔名
  • 定期備份你的資料夾

結語

現在你已經學會如何新增文章了！趕快試著創建你的第一篇文章吧。
