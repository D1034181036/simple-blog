如何新增一篇文章

2026-01-03

步驟一：複製 Template

  • 找到 posts/_template/ 資料夾
  • 複製整個 _template 資料夾

步驟二：重新命名資料夾

  • 格式：YYYY-MM-DD-文章標題
  • 範例：2026-01-03-how-to-create-a-post
  • 使用英文和連字號，避免空格和特殊符號

步驟三：修改文章內容

開啟新資料夾中的 index.html，檔案中有 <!-- 修改這裡 --> 注釋標示，依照指示填寫：

  • <title> 和 <h2>：填入文章標題
  • <time>：填入文章日期（格式：YYYY-MM-DD）
  • <article> 內容：撰寫你的文章內容

參考 HTML 常用標籤 來使用各種 HTML 元素

步驟四：更新文章列表

開啟 posts/index.html，在 <ul> 列表的最上方加入：

<li>
    <time>日期</time>
    <a href="資料夾名稱/index.html">文章標題</a>
</li>
